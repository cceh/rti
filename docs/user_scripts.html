Scripts¶

scripts/image_filer.py¶

File image runs into subdirectories.

Reads images from a camera’s memory card and files them into directories. All images with near enough timestamps get filed into the same directory. It also builds a sample.lp file in the directory containing a light position map for the PTM builder.

usage: scripts/image_filer.py [-h] [-v] FILE [FILE ...]

the image files to file
